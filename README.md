
LeaveRequest:

A complete end-to-end Leave Request Automation System built using:
Power Apps (Canvas App)
Power Automate (Approval Workflow)
SharePoint Online (Data Storage)
Microsoft 365 Integration

Features
Employees can submit leave requests through Power Apps
Automated approval process using Power Automate
SharePoint Online list to store and manage leave data
Email notifications for approvals and rejections
Manager dashboard to review and update requests
Real-time tracking of pending, approved, and rejected requests

Tech Stack
Power Apps (Canvas App)
Power Automate (Flows & Approvals)
SharePoint Online
Power Fx
Outlook / Microsoft 365
Power BI (optional for reporting)

Workflow Overview
Employee submits a leave request
SharePoint list item is created
Power Automate triggers the approval flow
Manager approves or rejects
Status updates automatically in SharePoint
Email notifications sent to employee
